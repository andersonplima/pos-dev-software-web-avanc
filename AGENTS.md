# AGENTS.md - JHipsterapp1

## Tech Stack

- **Backend**: Java 21, Spring Boot 4.0.3, Spring Cloud 2025.1.1
- **Frontend**: Angular 21.2.2, TypeScript 5.9.3
- **Build**: Maven, npm/Vitest
- **Generated by**: JHipster 9.0.0

## Commands

### Required Wrappers

Always use `./npmw` and `./mvnw` instead of `npm`/`mvn` - they install dependencies locally.

### Development

```bash
./npmw run backend:start       # Spring Boot dev server (port 8080)
./npmw run start             # Angular dev server with HMR (port 9000)
./npmw run watch             # Both concurrently with watch
```

### Testing

```bash
./mvnw verify                           # Backend tests (Spring Boot)
./npmw test                             # Frontend tests (Vitest, coverage)
./npmw run test:watch                    # Watch mode
./npmw run lint                         # ESLint check
./npmw run prettier:check               # Format check
```

### Build

```bash
./mvnw -Pprod clean verify              # Full production build (jar + frontend)
./npmw run webapp:prod                   # Frontend production build only
```

### API-First Development

```bash
./mvnw generate-sources                  # Generate API delegates from OpenAPI spec
```

## Architecture

- `src/main/java/` - Java backend source
- `src/main/webapp/` - Angular frontend source
- `src/main/resources/` - Config, swagger API definitions
- `src/main/docker/` - Docker compose files

**Entry point**: `com.anderson.jhipsterapp1.Jhipsterapp1App`

## Quirks & Requirements

- **JHipster Registry**: App requires JHipster-Registry at `http://localhost:8761` to start. Run first:

  ```bash
  docker compose -f src/main/docker/jhipster-registry.yml up --wait
  ```

- **Docker dependencies**: Some features need Docker:

  ```bash
  docker compose -f src/main/docker/services.yml up --wait  # MySQL + others
  ```

- **OpenAPI codegen**: Edit `src/main/resources/swagger/api.yml`, then run `./mvnw generate-sources` and implement generated delegate classes.

- **Component prefix**: Angular components/directives must use `jhi` prefix (e.g., `<jhi-component>`).

- **Java version**: Requires Java 21, Node >=24.14.0

## Lint Order

```bash
npm run prettier:format  # Prettier (pre-commit hook runs this)
npm run lint            # ESLint
./mvnw verify          # Backend tests + checkstyle
```
